Microsoft Azure SDK for .NET

Microsoft Azure에서 확장 가능한 웹 응용 프로그램 및 서비스를 빌드할 수 있도록 지원하려면 Microsoft Azure SDK for .NET을 설치하십시오.

Microsoft Azure SDK for .NET은 Visual Studio를 확장하여 Microsoft Azure에서 확장 가능한 웹 응용 프로그램 및 서비스를 만들고, 빌드하고, 디버깅하고, 실행하고, 패키징할 수 있도록 합니다.

참고: 이 페이지를 포함하고 있는 프로젝트는 필요한 도구 집합을 설치하기 위한 지침만 제공합니다.
